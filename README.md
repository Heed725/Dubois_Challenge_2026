## About the Challenge

A goal of Du Bois's exhibition in 1900 was to demonstrate the development of Black Americans since emancipation in 1865. One hundred and twenty-six years later, in 2026, this year's challenge focuses on the use of **maps and geography** in the collection â€” recreating Du Bois's cartographic plates using modern tools.

Each challenge has its own directory in this repository, including the original visual and associated data. Challenge directories contain data as **CSV files**, and geographic data as both **GeoJSON** and **shapefiles**.
